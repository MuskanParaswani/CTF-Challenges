Steganography Challenge
Description:
A dynamic steganography challenge where hidden data is embedded within media files such as images . The challenge ensures that each participant encounters a unique instance, with dynamically generated password and flags.
Participants must analyze the file using steganographic techniques and tools to extract concealed information. The encoding method and embedded data vary per session, increasing complexity and preventing pattern-based solving.
Skills Tested:
File analysis
Steganography tools (e.g., steghide, binwalk)
Hidden data extraction
Forensic investigation
